this plugin allows you to only enable/disable the specific XML-RPC methods you need.

== Description ==

If you want to enable the WordPress, Movable Type, MetaWeblog and Blogger XML-RPC publishing protocols, this allows you to only enable the specific methods you need. You can disable any other methods and make your blog less vulnerable for hackers/spammers.

== Installation ==

Enable the plugin and go to the option page "Disable XML-RPC Methods". There you can select the specific XML-RPC Methods you want to disable.
